## 在 OpenClaw 中配置 Kimi

OpenClaw 已经内置了 Moonshot AI 的支持，你可以选择两个不同的提供商：

### 1. Moonshot（通用对话）
适合日常对话、长文本处理、多模态等通用场景。

```json5
{
 env: { MOONSHOT_API_KEY: "***" },
 agents: {
 defaults: {
 model: { primary: "moonshot/kimi-k2.5" }
 }
 }
}
```

**服务地址**：`https://api.moonshot.cn/v1`

### 2. Kimi Coding（编程专用）
如果你希望 OpenClaw 主要用于编程、代码生成、技术任务，建议使用 Kimi Code 接口。

```json5
{
 env: { KIMI_API_KEY: "***" },
 agents: {
 defaults: {
 model: { primary: "kimi-coding/k2p5" },
 models: {
 "kimi-coding/k2p5": { alias: "Kimi K2.5" }
 }
 }
 }
}
```

**服务地址**（OpenAI 兼容）：`https://api.kimi.com/coding/v1`

## 推荐的使用方式

如果你希望 OpenClaw「学会用 Kimi」，可以这样做：

1. **让 OpenClaw 读取文档索引** — 将 `https://platform.moonshot.cn/docs/llms.txt` 喂给 OpenClaw，让它了解 Kimi API 的完整功能（Tool Calling、文件解析、长上下文等）。

2. **长上下文场景优先选 Kimi** — Kimi K2.5 支持 **256K tokens** 的上下文窗口，非常适合处理长文档分析、代码库理解等 OpenClaw 的复杂任务。

3. **编程任务用 Kimi Code** — 如果你用的是 OpenClaw 的 Coding Agent 功能，配置 `kimi-coding` 提供商会获得更好的代码生成和推理效果。

4. **利用 Tool Calling** — Kimi API 原生支持函数调用（Tool Call），你可以在 OpenClaw 中定义工具，让 Kimi 模型自主决策何时调用外部工具。
