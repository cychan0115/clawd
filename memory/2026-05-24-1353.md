### 3. 盘路计算逻辑（从 `panlu.js` 翻译）

页面JS源码中盘路计算逻辑如下，需要在程序中复现：

```python
# 从主队视角
def calc_result(h_score, g_score):
    if h_score > g_score: return 0  # 胜
    elif h_score == g_score: return 1  # 平
    else: return 2  # 负

def calc_pan_result(h_score, g_score, handicap):
    # 主队得分 - 盘口 > 客队得分 → 主队赢盘
    if h_score - handicap > g_score: return 0  # 赢
    elif abs((h_score - handicap) - g_score) < 0.001: return 1  # 走
    else: return 2  # 输

def calc_big_result(h_score, g_score, total_str):
    if not total_str: return -1
    t = float(total_str)
    total = h_score + g_score
    if total > t: return 0  # 大
    elif abs(total - t) < 0.001: return 1  # 走
    else: return 2  # 小

def reversal(v):  # 客队视角反转
    return 2 if v == 0 else (0 if v == 2 else 1)
```
