assistant: OpenClaw 有 **3 种语音方式**，看你想要什么场景：

---

## 1️⃣ **我发语音消息给你**（最实用）

我用 TTS 生成语音文件，直接发到对话里。

### 方案 A：ElevenLabs（质量高，要 API Key）
```bash
# 1. 安装 sag CLI
brew install steipete/tap/sag

# 2. 配置 API Key
openclaw config set env.ELEVENLABS_API_KEY "你的key"

# 3. 完成 ✅ 之后我说话就能变成语音发给你
```

**ElevenLabs 特点：**
- 声音自然，支持情绪标签 `[whispers]` `[excited]` `[laughs]`
- 多语言，中文也可以
- 有免费额度（每月 10k 字符）

### 方案 B：本地 TTS（免费，离线）
```bash
# 1. 让 OpenClaw 自动下载 runtime 和语音模型
openclaw skills install sherpa-onnx-tts

# 2. 配置路径到 openclaw.json
# 模型是英文的，中文效果一般
```

**本地 TTS 特点：**
- 完全免费，不需要 API key
- 隐私性好（不上云）
- 英文效果不错，中文支持有限

---

## 2️⃣ **实时语音通话**（像打电话）

需要配置电话服务商：
- **Twilio** / **Telnyx** / **Plivo**
- 我给你打电话，你接起来跟我语音对话
- 配置较复杂，需要国外手机号

```bash
openclaw config set plugins.entries.voice-call.enabled true
openclaw config set plugins.entries.voice-call.config.provider "twilio"
# ... 还要配 accountSid, authToken, fromNumber
```

---

## 3️⃣ **Discord 实时语音**（如果你在 Discord 用）

v2026.5.18 刚改进的：
- 我可以在 Discord 语音频道里实时说话/听你说
- 像 Siri 一样连续对话
- 需要 Discord bot 配置
